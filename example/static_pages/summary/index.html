Sommaire (summary)

Voir la page de documentation du composant sur le Système de Design de l’État

Documentation du tag

      
    Returns a summary item. Takes a list as parameter, with the following structure:

    items = [{ "link": "item1", "label": "First item title"}, {...}]

    **Tag name**::
        dsfr_summary
    **Usage**::
        {% dsfr_summary items %}

Exemples

Données

[{'label': 'First item title', 'link': 'link 1'},
 {'label': 'Second item title', 'link': 'link 2'}]

Résultat

Sommaire
  1. First item title
  2. Second item title
