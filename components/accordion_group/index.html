Accordéons – Groupe

Documentation du tag

Returns a group of accordion items. Takes a list of dicts as parameters (see the accordion tag for the structure of these dicts.)

Tag name: dsfr_accordion_group

Usage: {% dsfr_accordion_group data_list %}
